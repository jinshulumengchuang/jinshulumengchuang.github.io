Installion Guide:

On a device with docker installed:
sudo docker run -p 19010:8444 -p 19011:19011 -p 19012:19012 -d --restart always --name taichi --device /dev/dri/renderD128:/dev/dri/renderD128 --shm-size=1GB tubentubentu/taichi
You may change "renderD128" to another rendernode(GPU). Use command 'ls /dev/dri' to see the GPUs available.
If your device do not have gpu:
sudo docker run -p 19010:8444 -p 19011:19011 -p 19012:19012 -d --restart always --name taichi --shm-size=1GB tubentubentu/taichi-gpuless

Usage:

connect http://ip-of-your-device-with-docker:19011 to use the container.
http://ip-of-your-device-with-docker:19012 is for file managing.
You can enable kasmvnc's webp and udp, disable xfce4 compositing in settings to gain better exp.
Set Root password:
docker exec -it taichi bash
