CrossMusicPlayer

CrossMusicPlayer is a cross-device music player for web. A linux device is needed to use this proram (as server). CrossMusicPlayer can play both from you linux server and client devices.All plantforms have modern browsers should be supported as client.Chromium is recommend for client.

Installion Guide:

Visit https://github.com/jinshulumengchuang/CrossMusicPlayer to download the binary file.
tar xvpf path/to/tarball -C /opt
use systemd-nspawn to start the service:
systemd-nspawn --bind=path/to/the/floder/you/store/muisc:/var/www/html/music -UbD /opt/CrossMusicPlayer
connect iptoyoulinuxdevice:16634 to use the program
