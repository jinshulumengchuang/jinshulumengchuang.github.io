This software is based on Retroarch.(https://www.retroarch.com/)

The emulator needs time to prepare itself before run first time, please be patient.Also, chromium is the only supported plantform.
You may want to enter fullscreen mode for better experience.
Please quit the emulator in menu manually, otherwise you may lost your save.

Notes For Desktop Users(Windows/Linux/MacOS/FreeBSD/NetBSD/OpenBSD Desktop PC/Laptop):
Vist https://docs.libretro.com/guides/input-and-controls/ for key bindings.

Notes For Mobile Users(iPhone/iPad/Android Device/ Windows Tablet):
Multi-touch is not supported yet, it may be fixed in next release.
Turn on screen overlay and save the configuration when use it first time.
