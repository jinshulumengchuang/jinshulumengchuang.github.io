CrossVideoPlayer

CrossMusicPlayer is a cross-device music player for web. A device with docker installed is needed to use this proram (as server). CrossVideoPlayer can play both from you docker server and client devices.All plantforms have modern browsers should be supported as client.Chromium is recommend for client.

Installion Guide:

On the device with docker installed:
sudo docker run -p 9012:80 --restart always --name crossvideoplayer -v /path/to/your/videos:/var/www/html/Video -d tubentubentu/crossvideoplayer

Then connect http://ip-of-your-docker-device:9012 in your browser
