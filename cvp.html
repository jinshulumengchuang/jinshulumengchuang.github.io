CrossVideoPlayer

CrossVideoPlayer is a cross-device video player for web. A linux device is needed to use this proram (as server). CrossVideoPlayer can play both from you linux server and client devices.All plantforms have modern browsers should be supported as client.Chromium is recommend for client.

Installion Guide:

Apache2 and php is needed for this project
Clone the project:
git clone https://github.com/jinshulumengchuang/CrossVideoPlayer
Put the source to the folder of your site(eg./var/www/html(Debian) )
Don't forget to correct the premission!
