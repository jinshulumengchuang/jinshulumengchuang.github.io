Installion Guide:

On a device with docker installed:
sudo docker run -p 19000:8444 -p 19001:19001 -p 19002:19002 -p 19003:19003 -d --restart always --name retroarch-cross --device /dev/dri/renderD128:/dev/dri/renderD128 tubentubentu/retroarch-cross
You may change "renderD128" to another rendernode(GPU). Use command 'ls /dev/dri' to see the GPUs available.

Usage:

On desktop devices: connect http://ip-of-your-device-with-docker:19001 to use retroarch.
On mobile devices: connect http://ip-of-your-device-with-docker:19002 to use retroarch.
http://ip-of-your-device-with-docker:19003 is for file managing.
You can enable webp and udp function in settings to gain better preformance.
