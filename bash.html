BashFileM

BashFileM is a linux filemanager app writen by Bash, web tec, and a little Python, which supports remote accessing. BashFileM may run on Windows(Cygwin), IOS(ISH) ,MacOS ,and Android(Termux) since they all have bash, busybox python, and Chromium, but GNU/Linux is the only offically supported plantform.
This project needs Python3, Bash, Busybox and Chromium/Firefox.

Usage:


git clone https://github.com/jinshulumengchuang/BashFileM
cd BashFileM
./run.sh

Remote access:
http://ip-of-your-device:9866
If you are using a public network,please use run-local.sh instead.And use openssh to tunnel the service.
